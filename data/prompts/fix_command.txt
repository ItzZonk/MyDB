## Common Fixes

- "permission denied" → Add `sudo` prefix (if appropriate) or fix permissions
- "no such file or directory" → Check path, create missing directories
- "invalid option" → Check the correct flags for this tool version
- "merge conflict" → Suggest `git status` to see conflicted files
